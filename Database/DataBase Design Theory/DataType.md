## 데이터타입

- INT
  - 정수 저장
- BIGINT
  - 10억이 넘어가는 정수 저장시
- DECIMAL
  - 실수 저장
- VARCHAR
  - 문자 저장
  - 유연하게 저장
  - ex) varchar(5) 에 input “a”가 들어온 경우 a만 입력됨
- CHAR
  - 문자 저장
  - 유연하지 않게 저장
  - ex) char(5)에 input “a”가 들어온 경우 a\_\_\_\_가 입력됨
- LONGTEXT
  - 6만이 넘어가는 문자 저장
- DATETIME
  - 날짜/시간 데이터 저장
  - TIMEZONE 고려 X
- DATE
  - 날짜 데이터 저장
  - TIMEZONE 고려 X
- TIME
  - 시간 데이터 저장
  - TIMEZONE 고려 X
- TIMESTAMP
  - 날짜/시간 데이터 저장
  - TIMEZONE 고려
- TINYINT(1)
  - TRUE / FALSE
- FLOAT, DOUBLE, TEXT
  - 잘 사용하지 않음
