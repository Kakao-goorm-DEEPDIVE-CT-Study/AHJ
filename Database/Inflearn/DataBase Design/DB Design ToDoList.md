## 데이터 추출

- 닉네임, 비밀번호
- 제목, 내용, 날짜,시간,우선순위,카테고리
- 카테고리 이름, 카테고리 아이콘, 카테고리 색
- 완료유무
- 집중시간,각 앱별 집중시간
- 프로필 이미지,
- 앱 색, 앱 글자, 앱 언어

## 그룹핑

- user
  - 닉네임, 비밀번호, 프로필 이미지
- task
  - 제목, 내용, 날짜,시간,우선순위,카테고리,완료유무,사용자id
- category
  - 카테고리 이름, 카테고리 아이콘, 카테고리 색
- focus
  - 집중시간,날짜
