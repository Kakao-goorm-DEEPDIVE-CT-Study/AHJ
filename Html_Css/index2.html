블록 요소

블록 요소는 항상 새로운 줄에서 시작하며, 화면 전체 너비를 차지합니다.

이것은 블록 요소 입니다.
이것은 또 다른 블록 요소 입니다.

인라인 요소

인라인 요소는 줄 바꿈 없이 다른 요소와 같은 줄에 위치합니다. 컨텐츠 너비만큼만 차지합니다.

이것은 인라인 요소 입니다. 이것은 또 다른인라인 요소 입니다.

위 두 요소는같은 줄에 위치합니다.

인라인 블록 요소

인라인 블록 요소는 인라인 요소처럼 한 줄에 위치하지만, 블록 요소처럼 너비와 높이를 지정할 수 있습니다.

위 두 요소는 같은 줄에 위치하면서 각 요소의 너비가 다르게 설정됩니다.
